Les raccourcis

Executer votre dernière commande


  $ !!
  $ sudo !!

Créer un script avec votre dernière commande


  $ echo "!!" > script.sh

Réutiliser le dernier argument
de votre précedente commande


  $ cp script.sh /usr/bin
  $ !$

Executer votre dernière commande en remplaçant une valeur


  $ /usr/bin/script.exe
  $ ^exe^sh

Executer une commande
sans la sauvegarder dans l'historique


  $ <space>command

Retourner dans le répertoire précédent


  $ cd -

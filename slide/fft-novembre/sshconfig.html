Simplifiez-vous la vie avec un fichier de configuration SSH

Morgan Brunot

Connexion à un serveur

  • Plusieurs machines distantes
  • Nom d'utilisateur différents
  • Port d'accès non standard

Génération d'une clé SSH


  $ ssh-keygen -t rsa -C "brunot.morgan@gmail.com"

Envoyer la clé publique sur le serveur


  $ ssh-copy-id -i ~/.ssh/id_rsa.pub mbrunot@dev54.m6web.fr

La clé devrait se trouver dans le fichier `~/.ssh/authorized_keys` de votre serveur

Connexion à votre serveur distant


  $ ssh mbrunot@dev54.m6web.fr -p 22000

On peut se créer un alias !


  $ alias dev54='ssh mbrunot@dev54.m6web.fr -p 22000'
  $ dev54 # To connect

Ou alors configurer son ~/.ssh/config


  Host dev54
    HostName dev54.m6web.fr
    Port 22000
    User mbrunot

Connexion à votre serveur distant


  $ ssh dev54
